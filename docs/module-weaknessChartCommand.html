General Slash Command structure for fetching monster weaknesses and replying to the user with a visual representation.
Properties:
Name Type Description
name string The command name for the specific game.
description string The description of the command.
options Array.<Object> The available options (user types in shadow/monster they are curious about).
callback function The function handling the command execution. We call query functions to the database using the option given by the user then send that data off into the weakness chart function to then reply with a weakness chart embed.
