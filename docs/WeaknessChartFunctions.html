WeaknessChartFunctions

Functions create the weakness charts of a specified shadow/monster within Persona 3 Reload, Persona 3 Reload: Episode Aigis, Persona 4 Golden, Persona 5 Royal, & Metaphor: ReFantazio.

(async, static) createMetaphorWeaknessChart(data) → {Promise.<Buffer>}

Generates a Metaphor: ReFantazio weakness chart as an image buffer given a specific monster's weakness data within an object. The chart visually represents a monster's reaction to every element within the game.
